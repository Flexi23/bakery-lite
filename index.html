bakery-lite by Flexi23

html5 sound curve doodle

tiny sketch with a jquery soundcloud player and a very basic wave curve visualization.

Dependencies

soundmanager2, jquery and the soundcloud API

Install

XHR requires a HTTP host

Contact

Felix Woitzel (23flexi@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/Flexi23/bakery-lite

get the source code on GitHub : Flexi23/bakery-lite
